# Getting Started with ES|QL for Security Analysts

---

## Course Description

Explore the basics of Elasticsearch Query Language (ES|QL). Learn how to use ES|QL in Kibana to retrieve, filter, transform, and summarize security data to support real-world investigation workflows.

# Content Scope

## Track Learning Outcome (TLO)

Use ES|QL in Kibana to query, filter, transform, and analyze security event data to support basic investigation workflows.

---

<details>
<summary><strong>LO1 - Describe the purpose and basic structure of ES|QL and its role in querying security data in Kibana.</strong></summary>

**Challenge Title:** Explain ES|QL Structure and Purpose

**Topics Covered**
- What ES|QL is  
- ES|QL pipeline structure  
- Core command overview  
- How ES|QL supports investigations  
- Quick reference tools in Kibana  

**Learning Activity**  
Instructional video  

**Assessment**  
MC question  

</details>

---

<details>
<summary><strong>LO2 – Construct and execute a basic ES|QL query in Kibana to retrieve security event data from a specified index.</strong></summary>

**Challenge Title:** Construct and Execute a Basic Query

**Topics Covered**
- FROM, LIMIT, KEEP  
- Selecting multiple indices  
- Executing queries in Kibana  
- Viewing results  

**Learning Activity**  
Instructional video + guided lab  

**Assessment**  
CTFd challenge  

</details>

---

<details>
<summary><strong>LO3 – Apply ES|QL filtering and sorting commands to isolate relevant security events from a dataset.</strong></summary>

**Challenge Title:** Filter and Sort Security Data

**Topics Covered**
- WHERE, SORT, Boolean operators  
- Narrowing investigation scope  

**Learning Activity**  
Instructional video + guided lab  

**Assessment**  
CTFd challenge  

</details>

---

<details>
<summary><strong>LO4 – Use ES|QL transformation and aggregation commands to summarize security event data and identify patterns.</strong></summary>

- Aggregation functions (COUNT, SUM, AVG)

- Identifying spikes or trends  

**Learning Activity**  
Instructional video + guided lab  

**Assessment**  
CTFd challenge  

</details>

<details>

- Combining filtering and aggregation  
- Responding to new investigation clues

Capstone lab

Multi-step CTFd investigation
